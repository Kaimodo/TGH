## Installation
TGH is available on the [PowerShell Gallery](https://www.powershellgallery.com/packages/TGH/).

To Inspect:
```powershell
Save-Module -Name TGH -Path <path>
```
To install:
```powershell
Install-Module -Name TGH -Scope CurrentUser
```
