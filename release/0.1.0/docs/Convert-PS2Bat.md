# Convert-PS2Bat

## SYNOPSIS
Converts PowerShell scripts to batch files

## SYNTAX

```
Convert-PS2Bat [-Filepattern] <String> [<CommonParameters>]
```

## DESCRIPTION
This script converts one or more PowerShell scripts to .bat batch files.

## EXAMPLES

### BEISPIEL 1
```
./convert-ps2bat *.ps1
```
